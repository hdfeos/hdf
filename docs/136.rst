Release of H4toH5 Conversion Library and Tools Version 2.2.2

-------------------------------------------------------------------------------


Version 2.2.2 of the H4toH5 Conversion Library and Tools is now
available. This release has been tested with HDF5-1.8.11 and HDF 4.2.9.
See the conversion library and tools web page for complete details
regarding this release:

 http://www.hdfgroup.org/h4toh5/

The software can be obtained from the conversion library and tools
web page, as well as the HDF Downloads page:

 http://www.hdfgroup.org/downloads/index.html#h4h5

The conversion software consists of command-line utilities (h4toh5,
h5toh4) as well as a conversion library for converting individual
HDF4 objects to HDF5 objects.

Important new features and changes in this release include:

 o Improvements were added to configure for finding other libraries
 used by HDF5.

 o The h5toh4 conversion tool can now convert a fixed-size HDF5 string

.. code-block:: c

    in a compound datatype to an HDF4 char type.


Please see the Release file for complete details regarding this release:

 http://www.hdfgroup.org/ftp/HDF5/tools/h4toh5/src/h4h5tools-2.2.2-RELEASE.txt
