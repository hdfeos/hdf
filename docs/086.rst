Release of HDF Java Products 2.3
------------------------------------------------------------------------

The latest version of the HDF Java Products, version 2.3,
is available. This version was tested with HDF5-1.6.5 and
HDF 4.2r1. The HDF Java Products can be downloaded from the
following page:

 http://hdf.ncsa.uiuc.edu/hdf-java-html/

Version 2.3 includes bug fixes and a few new features.
Following is a list of the major changes in HDFView with this
release:

   * Support for HDF-SRB (Windows and Linux only)
   * Support for compound datatypes containing 2D arrays (or greater)
   * Create/display named datatypes in HDF5
   * Create link in hdf5
   * Improve ability to manipulate palette
   * Select row/column data as X axis for xy plot in the table view
   * Numerous bug fixes

For questions regarding these or other HDF issues, contact:

 hdfhelp@ncsa.uiuc.edu
