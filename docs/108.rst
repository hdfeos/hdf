Patches to HDF-Java Products 2.5

-------------------------------------------------------------------------------


There are three patches available for HDF-Java Products 2.5. The patches
have not been fully tested and are not available for all platforms

.. code-block:: c

   supported by HDF-Java Products version 2.5. For this reason, they do not
   replace it. However, they include new features and bug fixes which may
   be of interest to all users.


For a comprehensive list of the changes in each patch see:

 http://www.hdfgroup.org/ftp/HDF5/hdf-java/v25p3/README

The latest patch, Patch 03, contains the previous patches, and is located
here:

 http://www.hdfgroup.org/ftp/HDF5/hdf-java/v25p3/

It can be obtained from the HDF-Java Products home page at:

 http://www.hdfgroup.org/hdf-java-html/

New capabilities available with Patch 03 include the ability to show the
data pointed to by both object and region references. The patches also
include numerous bug fixes.
